# 웨딩 메이크업 아티스트 포트폴리오 기획서

## 📋 프로젝트 개요

**목적**: 웨딩 메이크업 아티스트의 작업물을 효과적으로 보여주고 예약 문의를 유도하는 포트폴리오 사이트

**타겟**: 결혼 예정인 예비 신부, 웨딩 플래너, 예식장 관계자

**핵심 가치**: 우아함, 전문성, 신뢰감, 세련됨

---

## 🎨 디자인 컨셉

### 색상 팔레트
- **Primary**: Soft Pink / Ivory / Beige (우아하고 여성스러운)
- **Secondary**: Gold / Rose Gold (고급스러움)
- **Accent**: Black / Charcoal (전문성, 가독성)
- **Background**: White / Cream (깔끔함)

### 폰트
- **제목**: 우아한 세리프 또는 모던한 산세리프
- **본문**: 가독성 좋은 산세리프

### 톤앤매너
- 깔끔하고 미니멀한 레이아웃
- 고품질 이미지 중심
- 부드럽고 우아한 애니메이션
- 모바일 우선 디자인 (인스타그램 유입 고려)

- 라이트박스 오픈: Scale + Fade

## 📊 Umami Analytics 설정 계획

### Umami란?
- **Google Analytics 대안**: 프라이버시 중심 오픈소스 웹 분석 도구
- **GDPR 준수**: 쿠키 없이 작동
- **간단한 통합**: Next.js에 쉽게 추가 가능

### 추적할 주요 지표
1. **페이지뷰**
   - 각 섹션별 조회수
   - 체류 시간

2. **사용자 행동**
   - 카카오톡 버튼 클릭 (가장 중요! 💬)
   - 포트폴리오 이미지 클릭
   - 스크롤 깊이

3. **트래픽 소스**
   - 인스타그램 유입
   - 직접 방문
   - 검색 엔진

4. **디바이스 정보**
   - 모바일 vs 데스크톱 비율
   - 브라우저 종류
   - 화면 크기

### 설정 방법

#### 1. Umami 계정 생성
- [umami.is](https://umami.is)에서 무료 계정 생성
- 또는 self-hosted 서버에 설치

#### 2. 웹사이트 추가
- Umami 대시보드에서 새 웹사이트 추가
- Website ID와 Script URL 받기

#### 3. Next.js 통합
```typescript
// app/layout.tsx에 추가
<Script
  src="https://cloud.umami.is/script.js"
  data-website-id="YOUR-WEBSITE-ID"
  strategy="lazyOnload"
/>
```

#### 4. 이벤트 추적 설정
```typescript
// 카카오톡 버튼 클릭 추적
umami.track('kakao-click');

// 포트폴리오 이미지 클릭 추적
umami.track('portfolio-view', { category: 'wedding' });
```

### 주요 이벤트 목록
- [ ] `kakao-click`: 카카오톡 문의 버튼 클릭
- [ ] `portfolio-view`: 포트폴리오 이미지 확대
- [ ] `section-view`: 각 섹션 진입
- [ ] `cta-click`: Hero CTA 버튼 클릭
- [ ] `service-view`: 서비스 카드 클릭

### 환경 변수
```env
# .env.local
NEXT_PUBLIC_UMAMI_WEBSITE_ID=your-website-id
NEXT_PUBLIC_UMAMI_URL=https://cloud.umami.is
```
